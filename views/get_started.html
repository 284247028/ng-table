Getting Started

Steps for getting started (example on right)

  1. Add references to jQuery and AngularJS.
  2. Add references to ngTable's javascript and css files.
  3. Where you declare your app module, add ngTable:
    angular.module('myApp',['ngTable']);
  4. In your html file within the controller where you plan to use ng-table, add:
    <table ng-table>...</table>
  5. We are going to define a basic style for table in style.css: 
    .ng-table {
    border: 1px solid #000;
}
  6. In your javascript file within the controller where you plan to use
    ng-table, lets add data that our table will use:
    7. $scope.users = [{name: "Moroni", age: 50},
                {name: "Tiancum", age: 43},
                {name: "Jacob", age: 27},
                {name: "Nephi", age: 29},
                {name: "Enos", age: 34}];
  7. Now create your table rows:
    ng-repeat="user in users"
  8. Below is the outcome of the table using the example on the right:
    {{user.name}} {{user.age}}

HTML

<html ng-app="myApp">  
<head lang="en">
    <meta charset="utf-8">
    <title>Getting Started With ngTable Example</title>  
    <script type="text/javascript" src="jquery-1.9.1.min.js"></script>
    <script type="text/javascript" src="angular.min.js"></script>
    <script type="text/javascript" src="ng-table.js"></script>
    <link rel="stylesheet" type="text/css" href="ng-table.css" />
    <script type="text/javascript" src="main.js"></script>
    <link rel="stylesheet" type="text/css" href="style.css" />
</head>
<body ng-controller="MyCtrl">
    <table ng-table class="table">
    <tr ng-repeat="user in users">
        <td title="Name">
            {{user.name}}
        </td>
        <td title="Age">
            {{user.age}}
        </td>
    </tr>
    </table>
</body>
</html>

Javascript

// main.js
var app = angular.module('myApp', ['ngTable']);
app.controller('MyCtrl', function($scope) {
    $scope.users = [{name: "Moroni", age: 50},
                     {name: "Tiancum", age: 43},
                     {name: "Jacob", age: 27},
                     {name: "Nephi", age: 29},
                     {name: "Enos", age: 34}];
});
